# Requirements

* yarn (Node.js)
* python 3

# How to run Vue Frontend

**Install and Run**

```bash
cd todo_front
yarn install
yarn serve
```

# How to run DRF Backend API

**Activate Virtual Environment**

```bash
python3 -m virtualenv venv
source venv/bin/activate
```

**Install and Run**
```bash
cd todo_api
pip3 install -r requirements.txt
python3 manage.py runserver
```
